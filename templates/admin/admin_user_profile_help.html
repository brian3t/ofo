How to Setup Email Message 

In your messages you can use some dynamic content.
To apply it you can use tags from the table below.
You can add these tags into your email body and email subject.
 
Reset Password Information
Tag Name  Tag Description 
 {reset_password_code} Code to reset the password 
 {reset_password_date} The latest date when user can reset the password 
 {reset_password_url} URL where user can reset the password
